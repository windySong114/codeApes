    • 伪类选择器：

      • :link：未访问的标签，用户未点击过标签应用的样式

      • :visited：已访问的标签，用户点击标签后访问了目标页面应用的样式

      • :active：激活状态，点击标签但未释放鼠标按钮时

      • :hover：鼠标悬停在标签上的样式

      • :focus：键盘或鼠标点击聚焦的样式

      html
      <!DOCTYPE html>
<html lang="en">
<head>
    <style type="text/css">
    /* 默认链接样式 */
    label {
        margin-right: 5px;
    }
    /* 未访问的链接 */
    label:link {
        color: #000000; /* 保持黑色 */
    }
    /* 已访问的链接 */
    label:visited {
        color: pink; /* 访问过的链接文字颜色为粉色 */
    }
    /* 鼠标悬停在链接上的样式 */
    label:hover {
        text-decoration: underline; /* 悬停时显示下划线 */
    }
    /* 激活状态（点击链接但未释放鼠标按钮时） */
    label:active {
        color: red; /* 激活状态时链接文字颜色为红色 */
    }
    input {
        width: 200px; /* 输入框宽度 */
        padding: 8px; /* 内边距 */
        border: 1px solid #ccc; /* 默认边框颜色 */
        border-radius: 4px; /* 边框圆角 */
    }
    /* 输入框获得焦点时的样式 */
    input:focus {
        border-color: dodgerblue; /* 获得焦点时边框颜色改为深蓝色 */
        box-shadow: 0 0 5px rgba(0, 0, 255, 0.5); /* 添加淡蓝色的阴影效果 */
        outline: none; /* 去掉默认的外边框 */
    }
    </style>
</head>
<body>
  <label for="username">名称：</label>
  <input type="text" id="username" name="username" placeholder="请输入名称">
</body>
</html>

    • 伪元素选择器：

      • :before：在元素的内容前面插入生成的内容

      • :after：在元素的内容后面插入生成的内容

      html
      <!DOCTYPE html>
<html lang="en">
<head>
    <style type="text/css">
    /* 列表项默认样式 */
    li {
      list-style-type: none; /* 去掉列表项默认的标志 */
      margin-bottom: 10px; /* 底部间距 */
      font-size: 16px; /* 字体大小 */
      position: relative; /* 相对定位 */
    }
    /* 在列表项前添加装饰性内容 */
    li:before {
      content: "\2022"; /* Unicode编码，表示圆点 */
      color: dodgerblue; /* 颜色 */
      font-weight: bold; /* 粗体 */
      margin-right: 5px; /* 右边距 */
    }
    /* 在列表项后添加装饰性内容 */
    li:after {
      content: "✓"; /* Unicode编码，表示勾号 */
      color: limegreen; /* 颜色 */
      font-weight: bold; /* 粗体 */
      margin-left: 5px; /* 左边距 */
    }
    </style>
</head>
<body>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
</body>
</html>

    • 属性选择器：

      • 标签名[属性名=值]：指定属性且属性值等于特定值的元素

      • 标签名[属性名~=值]：指定属性且属性值包含指定词

      • 标签名[属性名|=值]：指定属性且属性值与特定值完全相等或以特定值开头

      • 标签名[属性名^=值]：指定属性且属性值以特定值开头的元素

      • 标签名[属性名$=值]：指定属性且属性值以特定值结尾的元素

      • 标签名[属性名*=值]：指定属性且属性值包含特定子串的元素

      html
      <!DOCTYPE html>
<html lang="en">
<head>
    <style type="text/css">
    /* 需要完全匹配，需要和属性名一致 */
    input[type=text] {
        width: 100px;
    }
    /* 不需要完全匹配 */
    input[name~=user] {
        background-color: yellow;
    }
    /* 需要完全匹配，不需要和属性名一致*/
    input[name|=username] {
        border: 1px solid pink;
    }
    /* 匹配开头字符 */
    a[href^="http"] {
        color: #FFFF00;
    }
    /* 匹配结尾字符 */
    button[name$="mit"] {
        background-color: #4CAF50;
    }
    /* 匹配部分字符串 */
    button[class*=t] {
        color: white;
    }
    </style>
</head>
<body>
  <a href="http://">链接</a>
  <input type="text" name="username-name">
  <button class="btn1" name="submit">提交</button>
</body>
</html>

		h2 ~ p {
		    color: pink;
		}
    </style>
</head>
<body>
	<h2>标题</h2>
	<p>第一个段落，变为粉色</p>

	<div>
		<p>第三个段落，不会变为粉色</p>
	</div>
</body>
</html>
